Credly Badges Showcase

A Python project to fetch, organize, and display digital badges and issuing organizations from Credly.

Features

  • Fetches badges and organization info from Credly API
  • Generates Markdown and HTML tables of badges and organizations
  • Handles organization logos, links, and badge counts
  • Supports any number of organizations and badges

How It Works

  1. Configure your Credly API token and settings in settings.py.
  2. Run the Python scripts to fetch and process badge/organization data.
  3. Generate Markdown or HTML output for use in documentation or web pages.

Get Started

pip install -r requirements.txt
python services/credly.py
